# Blinkit Grocery Sales Dashboard

This project showcases a **Power BI dashboard** built on Blinkit grocery sales data.  
I performed **data cleaning, transformation, and feature engineering** to ensure accuracy and clarity, then created interactive visualizations to highlight key business insights.

---

## 🔑 Key Features
- **Overall Metrics**: Total sales ($1.20M), average sales ($141), number of items (8523), and average rating (3.9).
- **Sales by Fat Content**: Pie chart comparing Low Fat ($425.36K) vs Regular ($776.32K).
- **Item Type Analysis**: Bar chart showing top categories like Fruits & Vegetables, Snack Foods, and Household.
- **Outlet Establishment Trends**: Line chart tracking sales growth from 2010–2020, with peak sales of $205K.
- **Outlet Size Performance**: Donut chart comparing Small ($444.79K), Medium ($248.99K), and High ($507.90K) outlets.
- **Outlet Location Insights**: Bar chart showing Tier 3 ($472.13K) outperforming Tier 2 and Tier 1.
- **Outlet Type Comparison**: Table summarizing Grocery Stores vs Supermarket Types (sales, items, avg rating, visibility).
- **Interactive Filters**: Outlet Location Type, Outlet Size, and Item Type for customized analysis.

---

## 📊 Dataset
The cleaned dataset used for this dashboard is included in the `data/` folder.  
- **File:** blinkit_grocery_data_cleaned.csv  
- Contains grocery sales records with engineered metrics for outlet type, size, item categories, and ratings.

This dashboard helps identify:
